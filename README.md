## Technologies

### Project is Created With:

- Arduino IDE 2.1.1
- 1x ESP32 Development Board 30 pins
- 1x Potentiometer
- 5x Wires
- 1x USB-C Cable

## Setup

- Download Arduino IDE
- Wire ESP32 board as shown in 'Wiring Diagram'
- Input code in 'POTENTIOMETERESP32.ino' under 'Main'
- Enjoy!
